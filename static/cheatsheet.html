Special characters
. Any character except a newline
^ Beginning of string
$ End of string
\ Escape special character or signal a special sequence
A|B Matches either A or B, where A and B are arbitrary REs
Quantifiers
* 0 or more repetitions
+ 1 or more repetitions
? 0 or 1 repetitions
*?,+?,?? Non-greedy or minimal matching
{m} Exactly m repetitions
{m,n} m to n repetitions
{m,n}? Same as above but non-greedy
Character classes
[ ] Specifies a set of characters
[amk] Matches either a,m or k
[a-z] Matches any letter from a to z
[a-],
[-a]		 Matches a or -
[^abc] Match any character except a, b or c
Special sequences
\w Word characters
[a-zA-Z0-9_]
\W Any character that is not a word character, i.e. opposite of \w
\s Whitespace character
[ \t\n\r\f\v]
\S Any character that is not a whitespace character, i.e. opposite of \s
\d A digit, i.e [0-9]
\D Any character that is not a digit, i.e. opposite of \d
\A Matches only at the start of the string
\Z Matches only at the end of the string
\b Matches the empty string at the start or end of word
\B Matches the empty string when it is not at the start or end of a word
Groups
(...) Matches the RE inside the parentheses and creates a group
(?:...) Non-capturing group
(?P<name>...) A named capturing group
(?#...) A comment
(?=...) Positive lookahead
(?!...) Negative lookahead
(?<=...) Positive lookbehind
(?<!...) Negative lookbehind
(?aiLmsux) Specifies one or more flags. a for ASCII-only matching, i for ignore case, L for locale-dependent, m for multi-line, s for dot-matches-all, u for Unicode matching and x for verbose
